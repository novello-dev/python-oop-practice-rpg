# OOP RPG Combat Python

A simple turn-based RPG combat system built with Python, focused on practicing Object-Oriented Programming concepts.

The project uses classes for characters, weapons and health bars, allowing a hero and an enemy to fight using different weapons.

## Features

- Turn-based combat system
- Hero and enemy classes
- Weapon system
- Equip and drop weapon methods
- Health bar display in the terminal
- Basic damage calculation
- Colored health bars
- Simple game loop

## Technologies

- Python 3
- Object-Oriented Programming
